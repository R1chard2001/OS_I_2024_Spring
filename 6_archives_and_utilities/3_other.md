# Egyéb alkalmazások

## *time*

Idmérő alkalmazás. Megmérhetjük vele egy program futását.
```bash
time command
```

## *whereis*

Megnézhetjük vele egy program futtatási helyét vagy helyit. (egy program több helyen is előfordulhat)
```bash
whereis command
```

## *diff*

Két fájl különbségének kiíratása.
```bash
diff file1 file2
```
