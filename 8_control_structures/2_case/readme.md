# *case*

Kinézete:
```bash
case $variable in:
 case1): ...;;
 case2):
  ...
  ;;;
 ...
 *) ...;;
esac
```
Bármennyi ág lehet. a \*) eset az alapértelmezett.

## 1) hét napja

Egyszerű szkript, mejben egy számot megadunk, és az alapján kiírja a hét napjának nevét.
Át lehet írni, hogy a *date* programmal működjön.
